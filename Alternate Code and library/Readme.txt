Complete Code Setup:
Step 1: Setup the library (follow the "Library" folder "Library Setup".txt)
Step 2: Select the Board Manager
Step 3: Search the keyword "ESP32" into the searcch bar
Step 4: Find the "ESP32 by Espressif" with the version of "1.0.6" and install it
Step 5: Select the board to "ESP32 Dev Module" with necessary port for both ESP32 WiFi module
Step 6: First, compile the program.If it compliles correctly without error, then click the upload button

Note: Verify the Step 1 to Step 5 carefully.
